Kubeflow on Microk8s wth Ansible 
===

```shell script
ansible-playbook site.yml
```

### environment

| Field                   | Description                                | Default Value              |
|-------------------------|--------------------------------------------|----------------------------|
| playbook_http_proxy     | Specified http proxy for playbooks         |                            |
| playbook_https_proxy    | Specified https proxy for playbooks        |                            |
